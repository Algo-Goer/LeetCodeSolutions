给你一个字符串 s ，将该字符串中的大写字母转换成相同的小写字母，返回新的字符串。

<br>
示例 1：

输入：s = "Hello"
输出："hello"

示例 2：

输入：s = "here"
输出："here"

示例 3：

输入：s = "LOVELY"
输出："lovely"


提示：
- <= s.length <= 100
- s 由 ASCII 字符集中的可打印字符组成

<br>

来源：

力扣（LeetCode）

链接：

https://leetcode-cn.com/problems/to-lower-case

<br>


`核心代码模式`

```c
/*
 * @lc app=leetcode.cn id=709 lang=c
 *
 * [709] 转换成小写字母
 */

// @lc code=start

char *toLowerCase(char *s)
{
    for (int i = 0; i < strlen(s); i++)
    {
        if (isupper(s[i]))
        {
            s[i] += 32;
        }
    }
    return s;
}
// @lc code=end
```
